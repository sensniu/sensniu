## Featured Research

### Open-Vocabulary Gaze Object Prediction: Benchmark and Method

**ACM Multimedia 2026 Main Track — Accepted**

This work studies open-vocabulary gaze object prediction, aiming to localize
and recognize the objects attended to by humans while generalizing to
categories unseen during training.

- First student author and primary student contributor.
- Led the main student research work, including dataset construction,
  annotation pipeline development, model implementation, GIST design and
  implementation, and experimental evaluation.
- Introduced the **DiSG benchmark**, containing:
  - 13,041 images
  - 17,447 annotations
  - 86 gaze-object categories
  - 60 base categories and 26 novel categories
- Developed an OVGOP framework combining text-driven object discovery,
  gaze-guided spatial selection, and spatial-semantic disambiguation.
- Proposed **Gradient-Informed Selection Tuning (GIST)** to adapt
  task-sensitive parameters while preserving open-vocabulary generalization.

[View the OVGOP Project and DiSG Dataset](https://github.com/sensniu/ovgop)
